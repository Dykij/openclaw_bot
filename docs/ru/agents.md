# 🤖 Агенты и Роли

## Бригады

### Dmarket-Dev (3 роли)
- **dmarket-lead** — командир бригады, координация торговли
- **trader** — выполнение торговых операций
- **analyst** — анализ рынка и данных

### OpenClaw-Core (9 ролей)
- **master-lead** — главный координатор
- **architect** — проектирование архитектуры
- **senior-dev** — старший разработчик
- **code-reviewer** — ревью кода
- **test-strategist** — стратегия тестирования
- **test-runner** — выполнение тестов
- **devops** — CI/CD и инфраструктура
- **research-brigade-lead** — координация исследований
- **log-analyst** — анализ логов

### Research-Ops (3 роли)
- **research-lead** — координация исследований
- **deep-researcher** — глубокие исследования
- **paper-analyst** — анализ научных статей

### Standalone (5 ролей)
- **security-auditor** — аудит безопасности
- **ux-writer** — UX тексты
- **docs-writer** — документация
- **integration-tester** — интеграционное тестирование
- **performance-engineer** — оптимизация производительности

## Маршрутизация

Запросы автоматически маршрутизируются к подходящей бригаде через:
1. **Intent Classifier** — определение типа задачи
2. **SmartModelRouter** — UCB1 multi-armed bandit выбор модели
3. **Brigade Routing** — направление к нужной бригаде
