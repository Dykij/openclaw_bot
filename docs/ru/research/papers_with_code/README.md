## Подробные описания

### 1. ExGRPO: Learning to Reason from Experience

- **Авторы:** ExGRPO Team et al.
- **Источник:** Papers With Code
- **Дата:** 2026
- **Ссылка:** Papers With Code
- **Релевантность для OpenClaw:** 10/10

**Аннотация:**
ExGRPO расширяет стандартный GRPO (Group Relative Policy Optimization) путём обучения на собственном опыте. Модель использует прошлые удачные и неудачные попытки как дополнительный обучающий сигнал. Это значительно ускоряет обучение рассуждению и стабилизирует тренировку.

**Применение к OpenClaw Bot:**
- Прямое применение ExGRPO для обучения reasoning-навыков OpenClaw
- Использование истории диалогов бота как опыта для обучения
- Ускорение RL-цикла за счёт переиспользования прошлого опыта

### 2. BandPO: Bridging Trust Regions and Ratio Clipping for LLM RL

- **Авторы:** BandPO Team et al.
- **Источник:** Papers With Code
- **Дата:** 2026
- **Ссылка:** Papers With Code
- **Релевантность для OpenClaw:** 9/10

**Аннотация:**
BandPO объединяет преимущества trust-region методов и ratio clipping (PPO-стиль) для стабильного RL-обучения LLM. Метод использует вероятностные границы для контроля шага обучения. Достигается лучшая стабильность тренировки при сохранении скорости сходимости PPO.

**Применение к OpenClaw Bot:**
- Стабильный RL-тренинг на ограниченном железе без divergence
- Замена стандартного PPO на более надёжный BandPO для обучения бота
- Контроль размера шага обучения при fine-tuning на пользовательских данных

### 3. How Far Can Unsupervised RLVR Scale LLM Training?

- **Источник:** Papers With Code
- **Дата:** 2026

**Применение к OpenClaw Bot:**
- Автоматическая генерация reward-сигналов для обучения без разметки
- Масштабирование обучения бота через верифицируемые задачи (код, математика)
- Понимание ограничений RLVR для планирования обучающего пайплайна
